# 이미지 파일 안내

이 폴더에 다음 이미지 파일들을 추가해주세요:

## 필수 이미지

1. **logo.png**
   - 크기: 가로 200px 이상 권장
   - 형식: PNG (투명 배경 권장)
   - 용도: 상단 헤더 로고

2. **cd.png**
   - 크기: 500x500px 권장
   - 형식: PNG (투명 배경 권장)
   - 용도: 음악 플레이어 CD 이미지
   - CD 디스크 형태의 원형 이미지

## 선택 이미지

3. **download_icon.png** (현재 미사용)
   - 향후 다운로드 기능 추가 시 사용

## 임시 이미지 사용

이미지가 없어도 플레이어는 작동합니다. 
이미지가 로드되지 않으면 브라우저 기본 대체 텍스트가 표시됩니다.

온라인에서 무료 이미지를 찾으시려면:
- Unsplash (https://unsplash.com)
- Pixabay (https://pixabay.com)
- 또는 직접 디자인하신 이미지를 사용하세요
