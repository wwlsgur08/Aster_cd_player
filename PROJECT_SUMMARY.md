# 🎵 Aster Alarm - 전시용 매력 음악 플레이어

## ✨ 프로젝트 완성!

전시회에서 사용할 수 있는 매력 음악 플레이어가 완성되었습니다!

## 📦 포함된 파일

```
Aster_cd_player/
├── index.html              # 메인 HTML 페이지
├── styles.css              # 전체 스타일시트
├── server.py               # Python 개발 서버
├── start-server.ps1        # PowerShell 시작 스크립트
├── README.md               # 프로젝트 문서
├── QUICKSTART.md           # 빠른 시작 가이드
├── images/
│   ├── logo.svg           # 로고 이미지 (SVG)
│   ├── cd.svg             # CD 이미지 (SVG)
│   └── README.md          # 이미지 가이드
└── js/
    └── player.js          # 플레이어 JavaScript 코드
```

## 🚀 바로 시작하기

### 1단계: 서버 실행

**PowerShell에서:**
```powershell
.\start-server.ps1
```

**또는 Python으로:**
```bash
python server.py
```

### 2단계: 브라우저 접속

서버가 자동으로 브라우저를 열거나, 수동으로 다음 주소로 접속:
```
http://localhost:8000
```

## 🎮 주요 기능

### ✅ 구현된 기능

1. **자동 플레이리스트 로드**
   - Firebase에서 생성된 모든 음악을 자동으로 가져옵니다
   - 최신순으로 정렬되어 표시됩니다

2. **음악 재생 제어**
   - ▶️ 재생/일시정지
   - ⏮ 이전 곡
   - ⏭ 다음 곡
   - 자동 다음 곡 재생

3. **시각적 피드백**
   - 재생 중 CD 회전 애니메이션
   - 진행 바 표시
   - 현재/전체 시간 표시
   - 활성 곡 하이라이트

4. **음악 정보 표시**
   - 생성자 이름
   - 매력 조합 (레벨 포함)
   - 생성 시간

5. **키보드 단축키**
   - `스페이스바`: 재생/일시정지
   - `←`: 이전 곡
   - `→`: 다음 곡

6. **반응형 디자인**
   - 데스크톱, 태블릿, 모바일 모두 지원
   - 전시회 키오스크에 최적화

## 🎨 커스터마이징

### 색상 테마 변경

`styles.css` 파일 상단의 변수를 수정:

```css
:root {
  --primary-color: #6a5acd;      /* 보라색 → 원하는 색으로 */
  --secondary-color: #9370db;    /* 연보라 → 원하는 색으로 */
  --accent-color: #ffd700;       /* 금색 → 원하는 색으로 */
  --bg-dark: #0a0a1f;           /* 배경 → 원하는 색으로 */
}
```

### 레이아웃 조정

**플레이어 크기 변경:**
```css
.cd-container {
  width: 250px;   /* 기본: 200px */
  height: 250px;
}
```

**최대 너비 변경:**
```css
.main-container {
  max-width: 1000px;  /* 기본: 800px */
}
```

## 🐛 문제 해결

### 플레이리스트가 안 보여요
1. Firebase 연결 확인
2. 브라우저 콘솔(F12) 에러 확인
3. 인터넷 연결 확인

### 음악이 안 재생돼요
1. 음악 URL이 유효한지 확인
2. CORS 설정 확인
3. 브라우저 자동재생 정책 확인

### 이미지가 안 나와요
1. SVG 파일 경로 확인
2. 서버 로그에서 404 에러 확인
3. 브라우저 캐시 삭제 후 새로고침
