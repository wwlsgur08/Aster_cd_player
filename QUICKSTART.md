# 🎵 Aster Alarm 전시용 플레이어 - 빠른 시작 가이드

## 📋 준비사항

1. **이미지 파일 준비** (선택사항)
   - `images/logo.png` - 로고 이미지
   - `images/cd.png` - CD 이미지
   
   💡 이미지가 없어도 플레이어는 작동합니다!

## 🚀 실행 방법

### 방법 1: PowerShell 스크립트 사용 (권장)

```powershell
.\start-server.ps1
```

### 방법 2: Python 직접 실행

```bash
python server.py
```

또는

```bash
python -m http.server 8000
```

### 방법 3: Node.js 사용

```bash
npx http-server -p 8000
```

## 🎮 사용법

1. 서버가 시작되면 브라우저가 자동으로 열립니다
2. Firebase에서 플레이리스트가 자동으로 로드됩니다
3. 원하는 곡을 클릭하여 재생합니다

### 키보드 단축키
- `스페이스바`: 재생/일시정지
- `←` 왼쪽 화살표: 이전 곡
- `→` 오른쪽 화살표: 다음 곡

## 🎨 커스터마이징 가이드

### 1. 색상 변경
`styles.css` 파일의 `:root` 섹션 수정:

```css
:root {
  --primary-color: #6a5acd;      /* 메인 보라색 */
  --secondary-color: #9370db;    /* 연한 보라색 */
  --accent-color: #ffd700;       /* 금색 강조 */
  --bg-dark: #0a0a1f;           /* 어두운 배경 */
}
```

### 2. 플레이어 위치/크기 조정
`styles.css`의 `.main-container` 수정:

```css
.main-container {
  max-width: 1000px;  /* 더 넓게 */
  margin: 0 auto;
  padding: 20px;
}
```

### 3. CD 크기 변경
`styles.css`의 `.cd-container` 수정:

```css
.cd-container {
  width: 250px;   /* 더 크게 */
  height: 250px;
}
```

## 🔧 문제 해결

### 플레이리스트가 로드되지 않아요
- Firebase 연결을 확인하세요
- 브라우저 콘솔(F12)에서 에러 메시지를 확인하세요
- 인터넷 연결을 확인하세요

### 음악이 재생되지 않아요
- 음악 파일 URL이 유효한지 확인하세요
- CORS 설정을 확인하세요
- 브라우저에서 오디오 재생이 차단되지 않았는지 확인하세요

### 이미지가 표시되지 않아요
- `images/` 폴더에 올바른 파일이 있는지 확인하세요
- 파일 이름이 정확한지 확인하세요 (대소문자 구분)

## 📱 모바일/태블릿 사용

반응형 디자인이 적용되어 있어 모바일 기기에서도 사용할 수 있습니다.
전시회에서 태블릿을 사용하는 경우, 브라우저를 전체화면 모드로 전환하세요.

## 🎯 전시회 팁

1. **자동 재생 설정**: 첫 곡을 미리 로드해두세요
2. **화면 잠금 방지**: 기기 설정에서 화면 꺼짐 방지
3. **볼륨 조절**: 전시 공간에 맞게 적절한 볼륨 설정
4. **네트워크**: 안정적인 WiFi 연결 확인
