# 🎵 Aster Alarm - 전시용 매력 음악 플레이어

아이패드 가로모드에 최적화된 전시용 매력 음악 플레이어입니다.

## ✨ 주요 기능

### 🎨 아이패드 가로모드 레이아웃
- **왼쪽 패널**: LP/CD 플레이어
  - 회전하는 LP판 디스크
  - 재생/일시정지, 이전/다음 곡 컨트롤
  - 진행 바 및 시간 표시
  - 현재 재생 중인 음악 정보

- **오른쪽 패널**: 플레이리스트
  - 스크롤 가능한 음악 목록
  - 매력 카테고리별 색상 구분
  - 실시간 재생 상태 표시

### 🎵 데모 음악
7개의 샘플 트랙이 포함되어 있습니다:
1. **지민** - 정서적 안정 및 자기 인식 (침착함, 안정감, 긍정적)
2. **승현** - 유머감각 및 사교성 (유머 감각, 분위기 메이커, 사교적 에너지)
3. **수진** - 지적 호기심 및 개방성 (호기심, 창의성, 통찰력)
4. **민수** - 도덕성 및 양심 (정직함, 양심, 진정성)
5. **혜린** - 이해심 및 공감 능력 (다정함, 공감 능력, 배려심)
6. **태양** - 목표 지향성 및 야망 (목표 의식, 열정, 리더십)
7. **은서** - 성실성 및 책임감 (성실함, 책임감, 계획성)

### 🌈 매력 카테고리 시스템
- **이해심 및 공감 능력** (핑크색)
- **성실성 및 책임감** (청록색)
- **지적 호기심 및 개방성** (노란색)
- **정서적 안정 및 자기 인식** (녹색)
- **도덕성 및 양심** (파란색)
- **유머감각 및 사교성** (주황색)
- **목표 지향성 및 야망** (빨간색)

각 카테고리는 고유한 그라데이션 색상을 가지며, LP판 디스크 색상이 자동으로 변경됩니다.

## 🚀 실행 방법

### 1. Python 사용 (권장)
```bash
python server.py
```

### 2. PowerShell 스크립트
```powershell
.\start-server.ps1
```

### 3. 수동 실행
```bash
python -m http.server 8000
```

서버가 시작되면 브라우저에서 자동으로 `http://localhost:8000`이 열립니다.

## 📱 아이패드 설정 가이드

### 권장 설정
1. **화면 방향**: 가로모드 고정
2. **해상도**: 1024x768 이상
3. **브라우저**: Safari 또는 Chrome
4. **전체화면 모드**: Safari에서 전체화면 버튼 클릭 또는 F11

### 최적 화면 크기
- iPad Pro 12.9": 2732 x 2048 (가로: 2048 x 2732)
- iPad Pro 11": 2388 x 1668 (가로: 1668 x 2388)
- iPad Air/10.9": 2360 x 1640 (가로: 1640 x 2360)
- iPad 10.2": 2160 x 1620 (가로: 1620 x 2160)

## 🎮 사용 방법

### 마우스/터치
- **곡 선택**: 플레이리스트에서 곡 클릭/탭
- **재생/일시정지**: 재생 버튼 클릭
- **이전/다음**: ⏮ ⏭ 버튼 클릭
- **탐색**: 진행 바 클릭하여 원하는 위치로 이동

### 키보드 단축키
- `스페이스바`: 재생/일시정지
- `←`: 이전 곡
- `→`: 다음 곡

### 자동 재생
- 곡이 끝나면 자동으로 다음 곡 재생
- 마지막 곡이 끝나면 첫 곡으로 돌아감

## 🎨 커스터마이징

### 색상 변경
`styles.css` 파일의 `:root` 섹션에서 카테고리별 색상을 변경할 수 있습니다:

```css
:root {
  --empathy-from: #ec4899;
  --empathy-to: #be185d;
  /* ... 다른 색상들 */
}
```

### 데모 곡 변경
`js/player.js` 파일의 `DEMO_TRACKS` 배열을 수정하세요:

```javascript
const DEMO_TRACKS = [
  {
    id: '1',
    name: '이름',
    traits: [
      { charm_name: '매력이름', stage: 레벨 }
    ],
    duration: 초,
    audioUrl: '음악파일URL',
    createdAt: 타임스탬프
  },
  // ... 더 추가
];
```

### 레이아웃 조정
`styles.css`의 `.exhibition-layout`에서 그리드 비율 조정:

```css
.exhibition-layout {
  grid-template-columns: 45% 55%; /* 왼쪽:오른쪽 비율 */
}
```

## 🔧 문제 해결

### 음악이 재생되지 않아요
- 인터넷 연결 확인
- 브라우저 자동재생 정책 확인
- 오디오 파일 URL이 유효한지 확인

### LP판이 회전하지 않아요
- 재생 버튼을 눌렀는지 확인
- 브라우저 개발자 도구(F12)에서 에러 확인
- CSS 애니메이션이 활성화되어 있는지 확인

### 레이아웃이 깨져요
- 브라우저 화면을 가로모드로 전환
- 브라우저 줌 레벨을 100%로 설정
- 다른 브라우저에서 시도

### 색상이 제대로 표시되지 않아요
- 브라우저가 SVG를 지원하는지 확인
- 개발자 도구에서 CSS 오류 확인
- 캐시 삭제 후 새로고침
